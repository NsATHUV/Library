>[!info] 介绍
>JavaScript 是 Web 的编程语言。
> 所有现代的 HTML 页面都可以使用 JavaScript。
> JavaScript 非常易学。
> JavaScript 是互联网上最流行的脚本语言，这门语言可用于 HTML 和 web，更可广泛用于服务器、PC、笔记本电脑、平板电脑和智能手机等设备。

> JavaScript 是 web 开发人员必须学习的 3 门语言中的一门：
> 
> 1. **HTML** 定义了网页的内容
> 2. **CSS** 描述了网页的布局
> 3. **JavaScript** 控制了网页的行为
