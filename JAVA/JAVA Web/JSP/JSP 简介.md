### 简介

>[!info] JSP（Java Server Pages）是一种基于Java的服务器端网页技术，由Sun Microsystems（现为Oracle公司的一部分）开发，用于创建动态网页。JSP允许在HTML代码中嵌入Java代码，通过与Java EE（Java Platform, Enterprise Edition）框架的整合，为Web应用程序提供强大的功能支持。
