### 简介

>[!info] 
>JSTL是一个不断完成的开源的JSP标签库，通过JSTL可以替换jsp的脚本，实现流程控制。
>**JSTL（JavaServer Pages Standard Tag Library）** 是 Java EE 提供的一组标准标签库，用于简化 JSP 页面中的逻辑编写和数据处理操作。它将常见的编程任务封装为标签，使开发者能够使用 XML 样式的标签代替冗长的 Java 代码，从而提高代码可读性和开发效率。
