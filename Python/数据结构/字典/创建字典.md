### 字典的创建：

字典的创建使用花括号 `{}`，并使用冒号 `:` 分隔键和值。每个键-值对之间使用逗号 `,` 分隔。

```Python
my_dict = {'name': 'John', 'age': 25, 'city': 'New York'}
```

字典（Dictionary）是 Python 中一种灵活且强大的数据类型，用于存储键-值对（key-value pairs）。它是一种可变（mutable）的集合，允许存储任意类型的数据，包括整数、浮点数、字符串、列表、元组等。
