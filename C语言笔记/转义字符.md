# 常用的转义字符有：

 | 转义字符 | 含义                                                         |
 | -------- | ------------------------------------------------------------ |
 | `\"`     | 双引号                                                       |
 | `\'`     | 单引号                                                       |
 | `\\`     | 反斜杠                                                       |
 | `\n`     | ==换行==                                                         |
 | `\t`     | ==制表符==                                                       |
 | `\r`     | ==回车==                                                         |
 | `\b`     | 退格                                                         |
 | `\f`     | 换页                                                         |
 | `\v`     | 垂直制表符                                                   |
 | `\0`     | 空字符（NULL）编译器在处理字符串时，会为末尾加入该转义字符，表示字符串结束。 |
 | `\uXXXX` | Unicode 字符                                                 |
 | `\xXX`   | ASCII 字符                                                   |
